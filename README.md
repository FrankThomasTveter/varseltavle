# varseltavle
Warning tableau in REACT for meteorological warnings.

Download source:

      git clone https://github.com:/FrankThomasTveter/varseltavle.git

Initialise project:

      sudo apt install npm
      npm install
      npm start
      npm run build

Disable .# files by emacs, in .emacs add:

      (setq create-lockfiles nil)
